About Chicago Movie Parks

Updated for 2018

The City of Chicago has a summertime tradition of showing movies in the various parks all across. But, it isn't always easy to find movies showing near you. Use this website to find movies showing at parks near you.


Show Times and Data Accuracy

According to the Chicago Park District, movies start at dusk. I am using the sunrise-sunset.org API to determine the sunset time.

PLEASE CHECK THE CHICAGO PARK DISTRICT FOR CANCELLATIONS!

The database is published by the the Chicago Data Portal. However... the database is uploaded once and never updated afterwards. It is important that you use this website only as reference for what is available. If you'd like to see the dataset, click here
